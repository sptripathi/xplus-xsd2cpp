W3C Complaince

If you have gone through W3C XML-Schema specifications, then you would probably agree that it is very extensive. For the release being used, there may be some schema components that would not be supported. This section strives to document release-wise schema specification compliance of XmlPlus. If you find schema components in a xmlplus release that you found to be not supported by XmlPlus, but is not captured here, please let us know: email us at xmlplus.custodians@gmail.com.

This page captures schema components that are not supported in the latest release.
To view release-wise compliance including old releases, visit release-history.

XSD 1.0 Specifications:

  • XML Schema Part 2: Datatypes Second Edition
XSD 1.1 Specifications:
  • W3C XML Schema Definition Language (XSD) 1.1 Part 1: Structures
  • W3C XML Schema Definition Language (XSD) 1.1 Part 2: Datatypes
