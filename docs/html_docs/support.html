Report Bugs

For filing a bug, you could do one of the following:
  • file a bug at http://code.google.com/p/xplus-xsd2cpp/issues/list (preferred)
    OR
  • send an email to xmlplus.bugs@gmail.com, with details

For feedback or suggestions, email us at xmlplus.custodians@gmail.com.

Details expected in the bug/email(as appropriate):
  • Description of the problem and the use case that caused the issue
  • The OS name/release eg. Ubuntu 10.04
  • XmlPlus package information
    • if you are using a release tarball of XmlPlus, then the package release version.
    • if you are using hot svn source of XmlPlus, then svn revision number, at which your respositry is updated
  • The actual XML-Schema and XML files if possible, else samples derived from there, which are representative enough of the problem
  • Any other contextual information that you think is useful to debug the issue.
