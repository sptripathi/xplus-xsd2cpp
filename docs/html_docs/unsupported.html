Unsupported Stuff

If you have gone through W3C XML-Schema specifications, then you would probably agree that it is very extensive.
As we are at release 0.1, not everything from the specification is supported currently.
However, by release 0.3, we aim to support every item from following XML-Schema specifications/versions:
  • XML Schema Part 1: Structures Second Edition
  • XML Schema Part 2: Datatypes Second Edition


This section strives to document all those XML-Schema specification items that are not supported by XmlPlus. If you find something that you think is not supported by XmlPlus, but is not listed here, please let us know: email us at xmlplus.custodians@gmail.com.

Here is the unsupported components list:
  • complexContent inside complexType
  • simpleContent inside complexType
  • redefine
  • attributeGroup
  • group
  • notation
  • key
  • keyref
  • unique
  • wildcards and xsi-attributes in instance documents
  • following attributes on applicable schema components:
    • finalDefault
    • blockDefault
    • version
    • lang
    • final
    • block
    • nillable
    • abstract
    • substitutionGroup
