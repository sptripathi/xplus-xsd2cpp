Source Repository

The hot svn source is available at:
http://code.google.com/p/xplus-xsd2cpp

For checking out mainline(trunk):
    $ svn checkout http://xplus-xsd2cpp.googlecode.com/svn/trunk xmlplus_trunk

For checking out a release specific branch, eg. xmlplus-0.1:
    $ svn checkout http://xplus-xsd2cpp.googlecode.com/svn/xmlplus-0.1 xmlplus-0.1
